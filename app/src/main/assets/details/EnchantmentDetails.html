► 附魔是什麽？
  • 附魔是玩家的俗称，游戏中称为「强化装备」。附魔透过消耗装备的潜力值，可以在玩家製作的装备上附加上複数不同的能力，製作出多样化的装备
    所以我们说的附魔，其实就是客製化装备的能力

  • 也因为只能够在「玩家製作的装备」上附加能力，因此只可能在主手武器和衣服作附魔
    -
  • 习得锻冶大师技能树中的「强化装备」后，可以在【角色】→【技能】→【使用EX技能】→【锻冶大师】中进行【强化装备】
    -
  • 精华区中：【锻冶大师】精炼、附魔、製装相关，有牵魂梦的介绍影片

► 技能习得
  • 在图书馆学习【生活技能】中的【锻冶大师】技能树
  • 锻冶大师技能树中，【强化装备】和【正确的强化】会影响成功率、强化能力的数量及附魔数值的上下限，请点满！
  • 【素材加工】，不会分解材料的家伙还是个匠师吗！
    除非你要一直跑铁匠铺或切换角色来分解，不然还是点满吧

►角色等级与能力数值的影响
  • 等级会影响附魔的数值能力上下限，所以一般来说，附魔角色等级越高越好
    例如MDEF的可附魔数值上下限为：角色等级/10 （捨去小数）
    所以123级的角色可以附加【MDEF±12】的能力，160级的角色可以附加到【MDEF±16】

    但也不是每项能力的上下限都能附到这麽多，各能力都有其独有的数值上下限
    比如阻挡力，无论你是100级或160级的角色，都只能附加±3

► 潜力的计算
  • 玩家製作的装备会有潜力值。只有在当前潜力值大于0 且能力还没附满时才能附魔
    (如图，潜力值为0，无法附魔)

  • 每一次附加能力有所变动时，潜力值也会随之变动
  • 附加正向能力时消耗潜力值（耗潜）、负向能力时增加潜力值（退潜）
  • 在TEC255的情况下，退潜数值为耗潜数值的0.305倍（小数捨去）

    比如我若对一把三周弓附加【STR+2】，将会消耗潜力值10点；反之若附加【STR-2】，将会增加潜力值3点（现在潜力值47）



  • 各项能力有不同的基础潜力变动量，只附一条能力且并非两倍项目时，附魔时的潜力变动量为：
    潜力变动量 ＝ 基础潜力变动量 * 数值变动 * （退潜再乘以退潜倍率）

    比如我若对一把木剑附加【INT+6】，将会消耗潜力值5 * 6 = 30点；若附加【INT-4】，将会是5 * 4 * 0.305 = 6.1，捨去小数为增加6点潜力值

    比如我若对一把三周弓附加【HP自然回复%+1】，将会消耗潜力值20点；而同样的能力附加在衣服上只须消耗潜力值10点

    利用这一点，若在退潜时负向附加这些特定能力，将可以取得更多的潜力值
    比如对刚刚的三周弓作退潜，附加【HP自然回复%-3】，将可以取得10 * 3 * 0.305 * 2，得到18点潜力值；而同样的能力附加在衣服上只能取得10 * 3 * 0.305，得到9点潜力值
    -
  • 若对武器附加「属性觉醒」时，附加属性与铁匠版本相同，耗潜为十分之一
    比如我若对一把水之守护弓附加【火属性】，将会消耗潜力值100点；而附加【水属性】只须消耗潜力值10点
  • 属性觉醒不能退潜
  • 属性觉醒只能附上一种属性
  • 属性觉醒不能附在衣服上

  • 当装备的附加能力存在有同类项时，潜力变动量要乘以同类项倍率

    同类项倍率为：
    各单类同类项倍率的总和

    单类同类项倍率公式为：
    （单类同类项数目的平方 * 5）%

    作个表格方便查阅
    同类项数目 倍率
    2
    		 20%

    举例来说，现有一把三周年大剑，已附能力【暴击伤害+15】、【暴击+1】、剩馀潜力8
    我现在要对它作【暴击%+1】、【HP自然回复%- 7】、【MP自然回复-7】的附魔

    其中：
    【暴击伤害】、【暴击】、【暴击%】，同类3项，倍率45%
    【HP自然回复%】、【MP自然回复】，同类2项，倍率20%
    总和后得到同类项倍率为（45+20）% ＝ 65%

    则会：
    能力
    		 潜力
    		 备注
    暴击%+1 1 * 1 = 1 → 耗潜1 基础值1、变动1
    HP自然回复%- 7 10 * 7 * 2 * 0.305 = 42.7 → 退潜42 基础值10、变动7、两倍规则、退潜
    MP自然回复-7 10 * 7 * 2 * 0.305 = 42.7 → 退潜42 基础值10、变动7、两倍规则、退潜

    此次附魔得到136点潜力值

  • 附魔步骤的潜力公式为：
    附魔完潜力 = 附魔前潜力 + 潜力变动量 * (1 + 同类项倍率)

    参考前例，做完此次附魔，潜力值为 8 + 136 = 144

    请注意一定要按照步骤一步一步算，并捨去每一次计算的小数点，以免误差

  • 每个附魔数值的素材消耗为：基础值素材消耗量 * 附魔数值的平方
    例如目前手上有一把木剑，能力为【恨意值%+3】，我要将它提昇至【恨意值%+4】
    消耗素材为魔素，消耗量为33.5 * 4^2 = 536，也就是【恨意值%】从【+3 → +4】花536点魔素
    若要再提昇至【恨意值%+5】，消耗量为33.5 * 5^2 = 837.5 → 837，再花837点魔素
    -
  • 素材消耗没有退潜倍率、没有同类项倍率、没有两倍规则、没有十分之一规则
    只要一直累加总和就好，十分简单
    -
  • 所以承前例，若要直接赋予这把木剑【恨意值%+10】，也就是【恨意值%】从【+0 → +10】
    所需素材消耗就会是【+0 → +1】、【+1 → +2】、【+2 → +3】...【+9 → +10】的总和
    也就是33.5 *（1^2 + 2^2 + 3^2 + 4^2 + .... + 10^2）= 12897.5 → 12897
    -
    其实这是近似值，实际为12895，误差来自基础消耗量的引用，正确的基础消耗量为2579/77
    不过不像潜力值计算要求要很精确，消耗素材差个几点其实也没啥关系
    -
  • 每次这样加实在是太麻烦了，所以要简化一下 (′‧ω‧‵)
    -
  • 版友Aishimeth 给出了通式
    令附魔目标值为M、附魔现在值为N，附魔【N → M】的素材消耗为：
    基础值素材消耗量 * [ M (M + 1) (2M + 1) -  N (N+1) (2N + 1) ] / 6
    --
    简单明瞭，讚美数学！

    为了方便，M、N取绝对值了，如果真的有奇葩想附【-2→7】这种的话，正负请分开计算再加总
    也就是【0 → -2】和【0 → 7】加总 ，
    -
    举个例子，若要赋予一把枷锁切【迴避-7】【HP自然回复%-7】【MDEF%-7】，素材消耗为：
    【迴避】【0 → -7】 50 *（7 * 8 * 15 - 0 * 1 * 1）/6 = 7000 布料、基础值50
    【HP自然回复%】【0 → -7】 50 *（7 * 8 * 15 - 0 * 1 * 1）/6 = 7000 金属、基础值50
    【MDEF%】【0 → -7】 50 *（7 * 8 * 15 - 0 * 1 * 1）/6 = 7000 金属、基础值50
    共用去布料7000、金属14000
    -

► 成功率的计算
  • 以TEC255、点满「强化装备」和「正确的强化」的角色来说，该附魔步骤成功率为：
    〔附魔完剩馀潜力 / 现在潜力 * 230 + 130〕%
    （上限100%）
    -
    举个例子，我要将一把39潜的法杖附成【ATK+7%】【暴击+16】【暴击+16%】【暴击伤害+16】【HP自然回复-7%】【MP自然回复-7】的样子（A7、C1616、CD16）

    现在做到最后一步了
    已经附上【暴击伤害+12】【暴击+16】【暴击+16%】【HP自然回复-7%】【MP自然回复-7】
    正要附上【ATK+7%】【暴击伤害+12 → +16】，现在潜力109
    耗潜会是（10 * 7 + 3 * 4）* 1.65 = 135.3 → 135，馀潜-26
    成功率就是（-26 / 109 * 230 + 130）% = 75.14%

  • 观察公式可以发现，当附魔完剩馀潜力大于零时，成功率一定100%
  • 当附魔完剩馀潜力为负，〔附魔完潜力 / 现在潜力 * 230〕不小于-30，成功率也会是100%
    即〔附魔完潜力 / 现在潜力〕不小于-0.1304，成功率就会是100%
    -
  • 注意，当附魔完剩馀潜力为负，装备显示的潜力值会是0，然后就不能再附魔了
  • 由于只有在当前潜力值大于0时才能附魔，所以我们只有在最后一步附魔时才会让耗潜大于馀潜，不然后面的步骤会做不下去喔

► 附魔的步骤
  • 一件完成的附魔装备通常会经过许多次的附魔，我个人将这些附魔过程按注意事项分类为三个阶段，各阶段再有若干小步骤
  1. 预赋予阶段
    • 预赋予要在馀潜为正的前提下，尽量消耗装备的原始潜力值，减轻后赋予阶段的潜力负担
    • 如果有耗潜同类项，在这裡要先附上一些，可以在退潜阶段给出更多倍率
      -
  2. 退潜阶段
    • 退潜阶段要尽量取得更多的潜力值，因此请尽量在这裡用两倍项来退

  3. 后赋予阶段
    • 将剩馀附魔补上的阶段
      -
    • 在此阶段我们要尽量让最后一步的〔耗潜 / 馀潜〕不小于-0.1304
      否则成功率将低于100%
      -
    • 有些朋友可能已经发现了一件事，两倍项放在退潜阶段是毋庸置疑的，若退潜阶段与后赋予阶段有相同的同类项倍率，退潜阶段的退潜乘二可以得到更多潜力值；若后赋予阶段比退潜阶段有更高的同类项倍率。 要抵销两倍项的效益
      〔最后一步没乘倍率的馀潜〕+ 2 *      〔没乘倍率的退潜项〕 - 〔最后一步没乘倍率的耗潜〕
      要大于零
      -
    • 那麽非两倍的退潜项目是不是可以放在后赋予阶段的最后一步呢？这样不是能够透过减少耗潜来增加成功率吗？
      -
      我们可以做点数学验证，在同类项倍率相同的情况下：
      我们希望〔附魔完潜力 / 现在潜力〕越大越好，不要小于-0.1304最好
      所以在同类项倍率相同的情况下，尽可能将非两倍的退潜项放到退潜阶段比较好

      那麽在同类项倍率不同（后赋予阶段倍率较大）的情况下呢？
      条件允许的话，同类项倍率不同还是放到退潜阶段比较好的样子
      -
    • 所以不管同类项倍率如何，非两倍退潜项放退潜阶段都比放后赋予阶段好
      BUT....很多时候由不得你的，要是丢退潜阶段能力就附满了的话，此时也是不得不丢后赋予阶段的，就将耗潜多的退潜项留在退潜阶段吧
      -
  • 在最后一步前不要把能力都附满
  • 在最后一步前都要让潜力大于零
    -
  • 有个实用的技巧可以减低潜力值的消耗，也就是利用潜力值计算时会捨去小数点的特性，将一次高倍率的附魔拆成许多一步一步的附魔，藉此减轻倍率的影响
    -
    例如我现在要将一件49潜的衣服附上【暴击伤害+6%】【暴击伤害+16】【暴击+16】【暴击+16%】【MATK-7%】【魔法贯穿-3%】
    -
    在做完退潜阶段后馀潜76，此时倍率已达80%。若要将【暴击伤害+1→+16】一次附上，耗潜将达15 * 1.8 = 27，但若拆开一步一步附，即【暴击伤害+1→+2】、【暴击伤害+2→+3】、...、【暴击伤害+15→+16】，每次耗潜就会是1 * 1.8 = 1.8 → 1，做15次总耗潜只有15，很划算吧？
    -
  • 在预赋予阶段拆项附魔来省潜力值大概没什麽疑问，在退潜阶段不这麽做则是因为此阶段要求退更多潜力，拆项降低耗潜只会导致退出来的潜力比较少

    那麽后赋予阶段呢？
    答案是：算过才知道，有的能力拆项比较好，有的不拆比较好

► 实例
假设我现在要将一件59潜的衣服，附上双暴率、双暴伤，那麽具体步骤要怎麽做呢？
先列出基础耗潜
暴击、暴击% 1
暴击伤害 3
暴击伤害% 10

然后看同类项：暴击、暴击%、暴击伤害、暴击伤害%，有4项

在 预赋予阶段，1. 要尽量用掉原始潜力、2. 要堆倍率所以暴击、暴击%、暴击伤害、暴击伤害%都要用，附上的数值以让预赋予阶段结束时剩1是最好的调整暴击、暴击%、暴击伤害、暴击伤害%附上的的数值，让此四者耗潜为59-1 = 58
得到【暴击伤害%+1】五次50、【暴击伤害+1】两次6、【暴击%+1】一次1、【暴击+1】一次1共50+6+1+1 = 58
因此预赋予阶段的过程为：
能力 耗潜 说明
原始潜力59
暴击伤害%+5 10*5 = 50 基础10、变动5、同类项倍率0%、馀潜9（0%不用拆）
暴击伤害+1 3*1*1.2 = 3 基础3、变动1、同类项倍率20%、馀潜6
暴击伤害+1 3*1*1.2 = 3 基础3、变动1、同类项倍率20%、馀潜3
暴击%+1 1*1*1.2 = 1 基础1、变动1、同类项倍率20%、馀潜2
暴击+1 1*1*1.2 = 1 基础1、变动1、同类项倍率20%、馀潜1
预赋予阶段完成，馀潜1

进入退潜阶段，要尽量用两倍项来退衣服可以选择：
ATK、ATK%、MATK、MATK%、安定率%、物理贯穿%、魔法贯穿%、命中、命中%、对属性加强%

通常暴率暴伤的衣服是给物理职用，所以先拿掉物伤部份，剩：
能力 退潜
MATK 3*16 = 48，退14
MATK% 10*7 = 70，退21
魔法贯穿% 20*3 = 60，退18
命中 10*7 = 70，退21
命中% 20*3 = 60，退18
对属性加强% 5*14 = 70，退21
普通物职用，我们会选退MATK%和魔法贯穿%，这样副作用不大
但是注意，【暴击】、【暴击%】、【暴击伤害】、【暴击伤害%】已经四项退潜阶段时如果再同时放【MATK%】和【魔法贯穿%】，六条能力满我们会没办法附接下来的步骤所以很遗憾，其中一个要丢到后赋予阶段的最后一步附确定退【MATK%】和【魔法贯穿%】的话，就把退潜多的【MATK%】留在退潜阶段吧
如果将退【魔法贯穿%】(18)替换成退【命中】(21)或退【对属性加强%】(21)的话可以再降一些对衣服的潜力需求一般我们不会去退【对属性加强%】，退【命中】影响相对小，是可以考虑的方向
我们维持退【魔法贯穿%】继续往下做
因此退潜阶段的过程为：
能力 退潜 说明
馀潜1
MATK%-7 10*7*0.305*2 * 1.8 = 75 基础10、变动7、两倍项、同类项倍率80%、馀潜1+75
退潜阶段完成，馀潜76
同类项倍率最后乘、小数记得每次算完都要去掉
在后赋予阶段要补上剩下的附魔也就是【暴击伤害%+5 → +7】、【暴击伤害+2 → +16】、【暴击%+1 → +16】、【暴击+1 → +16】、【魔法贯穿% -3】

接下来我们需要计算各耗潜能力是否拆项来附，找出成功率最高的拆项方法
耗潜能力有4种，所以拆项方式有C(4,0)+C(4,1)+C(4,2)+C(4,3)+C(4,4) = 1+4+6+4+1 = 16种或者直接掏出恒等式，当n>=1、k = 0 to n，∑C(n, k)= 2^n，所以2^4 =16种
